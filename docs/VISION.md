# What is Arvus?

Arvus is a local AI operating layer that can:
- plan tasks
- control the PC
- execute tools
- remember context
- observe the screen
- assist with coding

## Core Principles

- Local-first
- Modular architecture
- Reliable execution
- Observability
- Tool-based intelligence
- Memory-driven reasoning

## Main Components

- Planner
- Router
- Memory
- Tool Engine
- Observer
- Executor
